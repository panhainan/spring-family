## SpringMVC学习笔记

### 一、SpringMVC数据绑定  
　　以下几种方式的代码请参见 `com.panhainan.web.DataBindController.java`  

#### （一）基于form表单和url传递参数方式的数据绑定   

　　1. 基本数据类型  
　　2. 包装类型  
　　3. 数组类型  
　　4. 简单对象（包括简单对象包含对象，即多层级对象）  
　　5. 同属性多对象  
　　6. 集合List类型（Set类型不太兼容，Map类型需要注意键值对，二者都不太常用）
#### （二）基于JSON格式和XML格式传递参数方式的数据绑定
　　1. JSON格式  
　　　　1.1 简单对象  
　　　　1.2 简单对象包含对象，即多层级对象  
　　　　1.3 对象数组  
　　　　1.4 集合List类型  
　　2. XML格式
